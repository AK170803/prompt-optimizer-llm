# 🧠 Prompt Optimizer (Local LLM)

A prompt engineering tool that transforms vague inputs into structured, high-quality prompts using a local LLM (Ollama).

## 🚀 Features

- ⚡ Prompt Optimization (structured format)
- 📊 Prompt Evaluation (rule-based scoring)
- 🧠 Explanation Mode (LLM-powered reasoning)
- 🔍 Diff View (see improvements clearly)
- 📦 Export as JSON

## 🛠 Tech Stack

- Python
- Streamlit
- Ollama (Local LLM)
